## Ghost Block Fix v1.0.0

Плагин фиксит проблему фантомных (ghost) блоков на игровом сервере (КОторые получается путем багнутого поршня или мода)
Он принудительно синхронизирует блоки рядом с игроком или порншнями + имеет тротлинг, дабы минимизировать нагрузку на сервер

### Совместимость
- Minecraft: 1.20.x, 1.21.x (Bukkit/Paper API)
- JDK: 21 и 25
