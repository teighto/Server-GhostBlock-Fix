## Ghost Block Fix v1.0.3

Плагин фиксит проблему фантомных (ghost) блоков на Paper/Bukkit. (баг при помощи поршней или модов)
Еще он имеет тротлинг, что-бы не перегружать сервер, так-как он принудительно синхронизирует блоки.

### Совместимость
- Minecraft: 1.20.x, 1.21.x, 26.1.x (Bukkit/Paper API)
- JDK: 21 и 25
